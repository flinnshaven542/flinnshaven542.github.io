## 🔍 Solving installation errors

Sometimes the installation halts or fails. Use these steps to resolve common conflicts.

### Check for missing updates
Windows 11 requires the latest security patches to run complex desktop applications. Go to Settings, select Windows Update, and click "Check for updates." Restart your PC if updates install.

### Run as administrator
Right-click the installer file. Select "Run as administrator." This grants the application permission to write files to protected system folders.

### Clean temporary files
Background processes often block new installations. Clear your temp folder:

1. Press the Windows key + R.
2. Type `%temp%` and press Enter.
3. Delete all files in the open window.
4. Try the installer again.

### Disable third-party security software
Antivirus programs sometimes identify new installers as threats. Temporarily disable your antivirus software. Ensure you re-enable it immediately after the installation finishes.

### Check folder permissions
Navigate to the desired installation folder. Right-click the folder and select "Properties." Click the "Security" tab and confirm that your user account has "Full Control" permissions.

## 🖥️ Typical error messages

You might see specific error codes during setup. Here is what they mean and how to act.

### Access Denied
The installer lacks permission to create the directory. Run the installer as an administrator as shown in the previous section.

### Disk Space Error
The drive contains insufficient space. Clear unnecessary files or move older data to an external drive. Verify you have at least 500 MB of space.

### Network Timeout
The download or installation process lost connection to the server. Check your internet connection. Connect via an ethernet cable if your Wi-Fi signal is weak.
